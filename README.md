## Projetos em destaque

### API REST com Node.js

API desenvolvida com **Node.js, Express e banco de dados SQL**, organizada em rotas, controllers e camada de conexão com o banco de dados.

O projeto tem como foco a construção de um backend funcional, estruturado e preparado para operações de cadastro, listagem, atualização e exclusão de dados.

### API REST com Flask

API desenvolvida com **Python, Flask e banco de dados relacional**, aplicando conceitos de requisições HTTP, rotas, manipulação de dados e integração entre aplicação e banco de dados.

O projeto reforça práticas de desenvolvimento backend, organização de endpoints e comunicação com o banco.

### API REST com Java Spring Boot

API desenvolvida com **Java e Spring Boot**, aplicando conceitos de controllers, services, repositories, entidades, rotas REST e integração com banco de dados.

O projeto tem como foco a construção de uma aplicação backend estruturada, seguindo uma organização em camadas e boas práticas de desenvolvimento.
